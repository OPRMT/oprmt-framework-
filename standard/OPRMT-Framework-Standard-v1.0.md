# OPRMT™ Framework Standard v1.0

## 1. Scope

Defines the official structure and requirements of the OPRMT™ framework.

---

## 2. Components

All prompts MUST include:

- Objective
- Parameters
- Results
- Method
- Tools

---

## 3. Rules

- Each component must be explicitly defined
- Missing components reduce effectiveness
- Structure must be maintained

---

## 4. Output Expectations

Prompts must produce:

- Clear outputs
- Structured responses
- Repeatable results
